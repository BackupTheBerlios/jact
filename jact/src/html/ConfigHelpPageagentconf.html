 Agentsconf

ACD distributes incoming calls to the agents of a Queue.
autologoff[String]Define autologoff times if appropriate. This is how long the phone has to ring with no answer before the agent is automatically logged off (in seconds). Please note that if this value is greater than the timeout value in your queue that agents will not be removed!
ackcall[String]Define ackcall to require an acknowledgement by '#' when an agent logs in over agentcallpark. Default is "yes".
wrapuptime[String]Define wrapuptime. This is the minimum amount of time when after disconnecting before the caller can receive a new call note this is in milliseconds.
music_class[String]Define the default musiconhold for agents musiconhold
custom_beep[String]An optional custom beep sound file to play to always-connected agents.
group[String]Group memberships for agents (may change in mid-file just)
agentsrecording[Agentsrecording]
agentsMap[Agents]


 Agentsrecording

This section is devoted to recording agent's calls The keywords are global to the chan_agent channel driver
recordagentcalls[Boolean]Enable recording calls addressed to agents. It's turned off by default.
recordformat[String]The format to be used to record the calls (wav, gsm, wav49) By default its "wav".
createlink[String]Insert into CDR userfield a name of the the created recording By default it's turned off.
urlprefix[String]The text to be added to the name of the recording. Allows forming a url link.
savecallsin[String]The optional directory to save the conversations in. The default is /var/spool/asterisk/monitor
